Τεχνητή Νοημοσύνη

Γλώσσα Προγραμματισμού: Python


Το project που μας ανατέθηκε αφορούσε στην ανάπτυξη αλγορίθμων μάθησης, του ID3, του Naive Bayes και του AdaBoost. Με τη βοήθεια αυτών των αλγορίθμων καθίσταται δυνατή η κατάταξη κειμένων σε δύο ξένες κατηγορίες (θετική και αρνητική). Χρησιμοποιήθηκε ένα σύνολο δεδομένων με 25.000 κριτικές από το IMDB. Με βάση αυτό το σύνολο μπορούσε κάθε κριτική να αναπαρασταθεί σαν ένα διάνυσμα από 0 ή 1. Το 0 αντιστοιχεί στην λέξη όπου δεν υπάρχει στην κριτική και το 1 όταν η λέξη περιλαμβάνεται στην κριτική. Οι λέξεις όμως επιλογόταν από ένα λεξιλόγιο. Το μέγεθος αυτού του λεξιλογίου εξαρτόταν από το ποσοστό επιτυχίας που θα είχε ο αλγόριθμος (έτσι αν επρόκειτο από accuracy 75% να πάμε σε ποσοστό 75,02% προσθέτοντας στο λεξιλόγιο 100 λέξεις και αναμένοντας το τέλος του αλγορίθμου για πολύ μεγαλύτερο χρονικό διάστημα τότε θα ήταν άσκοπο). Πρέπει να υπάρχει μία χρυσή τομή μεταξύ μεγέθους λεξιλογίου και χρονικής διάρκειας του αλγορίθμου.
Το project αυτό αναπτύχθηκε σε συνεργασία με τον Δημήτρη Αναστασόπουλο και τον Μιχάλη Δικαιόπουλο.

Παντομίμα - Δικό μου Project

Γλώσσα Προγραμματισμού: Java
Markup Language: XML


Αφορμώμενος από διάφορες εφαρμογές παντομίμας που κυκλοφορούν και που έχω και εγώ ο ίδιος χρησιμοποιήσει για διασκέδαση, αποφάσισα να δημιουργήσω τη δική μου εφαρμογή. Σε αντίθεση με την πλειονότητα αυτών των εφαρμογών που χρησιμοποιούν ένα μικρό σύνολο δεδομένων, αποφάσισα να πάρω ως είσοδο τεράστια dataset, προκειμένου να είναι σχεδόν μηδαμινή η πιθανότητα να ξανατύχει ο ίδιος τίτλος για αναπαράσταση. Δυστυχώς, όμως, αυτά τα dataset δεν διατίθονται στα Ελληνικά, με αποτέλεσμα να μην είναι δυνατή η αναπαράσταση των τίτλων στα Ελληνικά. Η τωρινή υλοποίηση παρέχει τους τίτλους στα Ελληνικά μέσω του εργαλείου αυτόματης μετάφρασης, η οποία δεν είναι και η καλύτερη δυνατή. Ωστόσο, πέρα από αυτά, υπάρχουν στην εφαρμογή διάφορες κατηγορίες για τίτλους, όπως παροιμίες, σειρές, ταινίες και τραγούδια, ενώ ταυτόχρονα υπάρχει και η δυνατότητα επιλογής των γύρων που θα έχει το παιχνίδι, του χρόνου για κάθε γύρο, καθώς και του αριθμού των ομάδων που θα λάβουν μέρος.
Το project αναπτύχθηκε αποκλειστικά από εμένα.

Γλώσσα Προγραμματισμού: Java
Markup Language: XML
